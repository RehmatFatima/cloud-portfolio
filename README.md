# Cloud Portfolio Hosting
This is my personal portfolio hosted on **Firebase Hosting**, demonstrating cloud deployment and static site hosting. It showcases my skills in HTML, CSS, and basic cloud concepts.

## Live Portfolio
You can view the live portfolio here:  
[https://rehmat-cloud.web.app](https://rehmat-cloud.web.app)

## Project Features
- Static website hosted on Firebase
- Uses Firebase Hosting for cloud deployment
- Clean and minimalistic design
- Responsive layout for desktop and mobile

## Technologies Used
- **Firebase Hosting**
- **HTML** for structure
- **CSS** for styling
- **Firebase SDK (optional)** for future cloud features
